[변수]
    1. 변수 : 하나의 데이터를 저장할 수 있는 메모리 공간 (논리적 공간)
    2. 변수명 : 메모리 공간의 상징적인 이름
    3. 변수종류
        1. 지역변수 : 특정한 { } 안에서 선언된 변수 (ex : 메서드{} , if{} , for{})
            -> 선언된 중괄호 밖으로 호출이 불가능
        2. 전역변수 : 선언될때 static 키워드가 붙은 변수
            -> 프로젝트 내 모든 클래스에서 호출이 가능
        3. 매개변수 : 함수 호출 시 인자값을 함수에 대입하는 변수
            -> 중매(매) : 매개변수, 인수값 <--> 변수(매개변수)
            -> 함수 내에서만 사용 가능한 변수 , 인수(들어가는 수)
        4. 멤버변수 : 클래스 안에 존재하고 함수 밖에 존재하는 변수
            -> 객체가 생성될때 같이 생성되는 변수 , 객체를 통해 접근가능한 변수
            -> 2가지 :
                1. 다른 클래스를 사용할 때 지정한 멤버변수를 호출시
                    클래스 변수 = new 클래스()
                    변수.멤버변수;
                2. 같은 클래스 내 멤버변수 호출 시
                    this.멤버변수

                주의할 점 : 변수는 참조라는걸 이용해서 하나의 값을 기차처럼 연결/참조하는 구조
                -> 메모리 효율성 극대화
                    Example1 변수 = new Example1();
                        1. 변수는 하나의 객체를 참조한다.
                        2. 객체는 각 멤버변수들을 참조한다.
                        3. 각 멤버변수들은 각 값/객체를 참조한다

                        변수 -> Example1객체 -> 멤버변수1
                                            -> 멤버변수2
                                            -> 전역변수

    기본타입과 참조타입을 구분하는 방법
        1. 기본타입 -> 리터럴 저장 int,double,boolean,char...
        2. 참조타입 -> 주소 저장 , 배열[] , 클래스 , 인터페이스 , 열거...
